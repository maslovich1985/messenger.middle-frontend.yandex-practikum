## В проекте применял технологии:
* Node JS
* Express
* Шаблонизатор handlebars
* Препроцессор Less
* Сборщик Parcel

##Установка/скрипты:

* npm run start - запуск проекта локально (Express, http://localhost:3000)
* npm run dev - запуск проекта локально (Parcel)
* npm run build - сборка проекта
